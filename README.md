# Love Calculator 🥰💕

A fun, simple web application that calculates the love percentage between two people based on their names. Enter names and get an instant result with playful compatibility advice!

## ✨ Features
- **Quick Calculation**: Generates a random love percentage (1-100%) on button click.
- **Smart Messages**: 
  - <30%: Not a great match! 😕
  - 30-79%: Somewhat okay, keep trying! 😌
  - 80%+: Great match, don't look elsewhere! 😍
- **Input Validation**: Alerts if names are empty.
- **Responsive Design**: Pink romantic theme, works on desktop/mobile.
- **No Dependencies**: Pure HTML/CSS/JS – instant play!

## 🛠️ Tech Stack
- **HTML5**: Structure and form.
- **CSS3**: Styling with flexbox, shadows, hover effects.
- **Vanilla JavaScript**: Logic for validation, random generation, dynamic results.

## 🚀 How to Run
1. Clone/open this project folder.
2. Double-click `index.html` to open in your browser (Chrome, Firefox, etc.).
3. Or in VSCode: Right-click `index.html` → \"Open with Live Server\".

No installation needed!

## 📖 Usage
1. Enter **Your Name** (e.g., Alice).
2. Enter **Your Crush's Name** (e.g., Bob).
3. Click **Calculate**.
4. View the love percentage and fun message!

**Example Result**:  
`Alice And Bob's Love Percentage Is = 85`  
`Great Match! Don't Search Others. 😍`
